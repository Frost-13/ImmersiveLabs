through the input file line by line and call the encryption function and then the letters in the line get matched 
to the letters in mapping file and encrypts it accordingly. decryption is similar but the opposite and is called on mode 2.


to run the program just type make it will compile then type ./test_encryption.sh     or     ./test_decryption.sh
